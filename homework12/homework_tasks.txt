В консоли
0. Создать массив с 50 записями: строками с числами ('123') и строками с текстом ('abc'), на его основе создать новый массив, где будут все числа из первого и число 8 вместо текста.

1. Напишем функцию, которая бы сортировала массив, и возвращала занчение, сколько объектов изменили свой порядок. Пример: mySortCount([0, 2, 1]) # => 2

2. Напишите программу, которая выводит на экран числа от 1 до 100. При этом вместо чисел, кратных трем, программа должна выводить слово «Fizz», а вместо чисел, кратных пяти — слово «Buzz». Если число кратно и 3, и 5, то программа должна выводить слово «FizzBuzz»


В браузере
3. Создадим "класс" ButtonManager, на странице сделаем кнопку. При нажатии на нее должно выводить, сколько раз она была нажата.

4. Сделаем input с адресом имейла и кнопку. При нажатии на кнопку нужно проверять, есть ли в отправленном email символ "@". Выводить результат в консоль.

5. При загрузки страницы необходимо находить все картинки, проверять, есть ли у них alt и title. Выводить в div результат проверки в формате: сколько картинок, % без alt, % без title.

6. При нажатии на кнопку "добавить" происходит создание на странице div со случайным (1 < x < 5) количеством параграфов p с текстом LoremIpsum. Страницу можно очистить при нажатии кнопки "очистить".
